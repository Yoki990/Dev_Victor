-Créer repo sur GitHub (à laisser en public) (si privé invite de collaboration à envoyer aux personnes invitées)



-Ouvrir Git Bash

-$ cd *Nom du dossier* (déplacement vers un dossier)

-$ cd .. (déplacement au dossier parent)



-$ git clone + lien du GitHub

-$ git init (création du dossier .git)

-$ git remote add origin + lien GitHub (permet de lier GitHub avec le dossier utilisé)

-$ git status (permet de voir les fichiers untracked)

-$ git add . (permet de préparer les fichiers untracked au commit. ET À CHAQUE MODIFICATION DU CONTENU DU DOSSIER !!!)

-$ git commit -m "*Nom de commit*" (sauvegarder localement)

-$ git push (permet d'envoyer les fichiers du dossier au GitHub)

-$ git nano *Nom du fichier* (créer et modifier le document) (CTRL + O pour enregistrer, CTRL + X pour sortir du nano)

-$ git pull (permet de mettre les fichiers à jour du GitHub au dossier)
